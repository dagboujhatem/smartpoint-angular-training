Ng-Model example 1: simple format

Ng-Model example

{{ 'Typed text: ' + value}}

NOTE:

Dans ce cas, si vous utilisez le [(ngModel)]="value", il faut ajouter l'attribut name à l'input ayant le même valeur que ngModel.

      Exemple:
      
      <input type="text" class="form-control mb-3" name="value" [(ngModel)]="value" >
