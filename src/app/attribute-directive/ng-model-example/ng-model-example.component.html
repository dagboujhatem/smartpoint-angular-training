NOTE:

Si vous utilisez le [(ngModel)]="value" à l'intérieur d'une balise de formulaire <form > .... < /form > alors une erreur sera affiché dans le console comme suit :

Solution 1:
      il faut ajouter l'attribut name à l'input ayant le même valeur que ngModel.
      
      <input type="text" class="form-control mb-3" name="value" [(ngModel)]="value" >
Solution 2:
il faut ajouter la directive [ngModelOptions]="❴ standalone: true }" . 
      <input type="text" class="form-control mb-3" [ngModelOptions]="❴ standalone: true }" [(ngModel)]="value" >
